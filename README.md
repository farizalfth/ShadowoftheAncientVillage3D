# Shadow of the Ancient Village 3D

Shadow of the Ancient Village 3D is a third-person adventure game developed with Unity. Players explore an abandoned ancient village filled with mystery, hidden paths, and environmental challenges. The game focuses on exploration, immersive environments, and storytelling while uncovering the secrets left behind by a forgotten civilization.

## Features

- Third-person character controller
- Ancient village environment
- Exploration-based gameplay
- Interactive objects
- Atmospheric lighting and environment
- Unity 3D project

## Built With

- Unity 6
- C#
- Unity 3D Game Kit
- SkySeries Freebie

## Getting Started

### Requirements

- Unity 6 or newer
- Git

### Installation

1. Clone the repository

```bash
git clone https://github.com/farizalfth/ShadowoftheAncientVillage3D.git
```

2. Open the project using Unity Hub.

3. Wait until Unity finishes importing all assets.

4. Open the desired scene inside:

```
Assets/Scenes
```

5. Press **Play** to run the game.

## Gameplay

The player explores an abandoned ancient village surrounded by forests and mysterious ruins. Throughout the journey, players will discover hidden locations, interact with the environment, and uncover the story behind the lost civilization.
